# 🌍🍽️ WorldCuisineAgent

An AI agent built using SmolAgents and Hugging Face that suggests themed menus inspired by cuisines from around the world.

## Features

- Country-based menu recommendations
- Party theme suggestions
- Appetizer recommendations
- Main course recommendations
- Dessert recommendations
- Beverage recommendations
- Fun facts about cuisines

## Supported Countries

- India
- Japan
- Italy
- Mexico
- China
- France
- Spain
- Thailand
- South Korea
- Turkey
- Greece
- Germany
- Brazil
- USA
- United Kingdom

## Example

```python
agent.run("Suggest a Japanese menu")
```

### Sample Output

Country: Japan

Theme: Tokyo Night Festival

Appetizer: Edamame

Main Course: Sushi Platter

Dessert: Mochi

Beverage: Matcha Tea
